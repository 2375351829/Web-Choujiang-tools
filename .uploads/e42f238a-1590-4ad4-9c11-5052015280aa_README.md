## 🚀 快速开始

### 1. 安装依赖

```bash
cd web-choujiang-tools
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

### 3. 访问系统

- 管理后台: http://localhost:3000
- 抽奖展示: http://localhost:3000/display.html
