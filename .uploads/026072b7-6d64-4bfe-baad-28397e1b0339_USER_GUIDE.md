## 👥 人员管理

### 1. 添加人员

**步骤 1**: 点击左侧菜单「人员管理」

**步骤 2**: 点击「添加人员」按钮

**步骤 3**: 填写信息
```
┌────────────────────────────────────────┐
│         添加人员                        │
├────────────────────────────────────────┤
│  姓名 *    │ [输入姓名           ]      │
│  邮箱      │ [输入邮箱           ]      │
│  电话      │ [输入电话           ]      │
│  中奖权重  │ [1                  ]      │
│            │ (权重越高，中奖概率越大)     │
├────────────────────────────────────────┤
│            [取消]  [确认添加]           │
└────────────────────────────────────────┘
```

**步骤 4**: 点击「确认添加」

**说明**:
- `*` 标记的字段为必填
- 邮箱电话可为空，不影响参与
- 权重默认为1，数值越高被抽中概率越大

### 2. 批量导入人员

**适用场景**: 参与者数量较多时（如100人以上）

**步骤 1**: 准备Excel文件

**Excel格式要求**:
| 列名 | 必填 | 说明 |
|------|------|------|
| name | ✅ | 姓名 |
| email | ❌ | 邮箱 |
| phone | ❌ | 电话 |

**示例**:
```excel
| name     | email              | phone      |
|----------|--------------------|------------|
| 张三     | zhangsan@email.com | 13800138000|
| 李四     | lisi@email.com     | 13900139000|
| 王五     | wangwu@email.com   |            |
```

**步骤 2**: 点击「批量导入」

**步骤 3**: 选择Excel文件

**步骤 4**: 等待导入完成

**步骤 5**: 查看导入结果

```
导入成功: 95人
已存在: 5人 (跳过)
总计处理: 100人
```

### 3. 编辑人员

**步骤 1**: 在人员列表找到目标人员

**步骤 2**: 点击操作列的「编辑」按钮

**步骤 3**: 修改信息后点击「保存」

### 4. 删除人员

**步骤 1**: 在人员列表找到目标人员

**步骤 2**: 点击操作列的「删除」按钮

**步骤 3**: 确认删除操作

**注意**: 删除后不可恢复，关联的抽奖记录会保留

### 5. 黑名单管理

**功能**: 将指定人员排除在抽奖之外

**步骤 1**: 在人员列表找到目标人员

**步骤 2**: 点击「黑名单」开关

**状态说明**:
| 状态 | 参与抽奖 | 说明 |
|------|---------|------|
| 正常 | ✅ 可以 | 默认状态 |
| 黑名单 | ❌ 不能 | 被排除 |

### 6. 查询筛选

**可用筛选条件**:
- 按状态：全部/已中奖/未中奖
- 按黑名单：全部/正常/黑名单
- 搜索：按姓名/邮箱/电话搜索

## 🎁 奖品管理

### 1. 添加奖品

**步骤 1**: 点击左侧菜单「奖品管理」

**步骤 2**: 点击「添加奖品」按钮

**步骤 3**: 填写奖品信息
```
┌────────────────────────────────────────┐
│         添加奖品                        │
├────────────────────────────────────────┤
│  奖品名称 *  │ [输入名称           ]    │
│  奖品描述    │ [输入描述           ]    │
│  奖品图片    │ [选择图片上传       ]    │
│  奖品数量 *  │ [1                  ]    │
│            │ (可抽取次数)              │
├────────────────────────────────────────┤
│            [取消]  [确认添加]           │
└────────────────────────────────────────┘
```

**说明**:
- 奖品数量决定该奖品可以被抽中多少次
- 图片支持jpg/png/gif格式，最大10MB

### 2. 编辑奖品

**步骤 1**: 在奖品列表找到目标奖品

**步骤 2**: 点击「编辑」按钮

**步骤 3**: 修改信息后点击「保存」

### 3. 删除奖品

**注意**: 删除奖品会同时删除关联的场次配置

### 4. 奖品属性说明

| 属性 | 说明 | 示例 |
|------|------|------|
| name | 奖品名称，会展示给观众 | "一等奖-iPhone" |
| description | 详细描述 | "iPhone 15 Pro 256GB" |
| image | 奖品图片 | - |
| draw_count | 可抽次数 | 3（表示可中3次） |

### 2. 抽奖操作流程

#### 完整抽奖流程

**步骤 1**: 确认场次配置
- 确保场次已添加参与者和奖品

**步骤 2**: 输入抽奖参数
- 场次ID：输入要抽奖的场次编号
- 奖品ID：输入要抽取的奖品编号

**步骤 3**: 点击「开始抽奖」
- 动画开始播放
- 参与者名字开始滚动

**步骤 4**: (可选) 点击「暂停」
- 动画暂停在当前状态
- 便于观察或准备

**步骤 5**: 点击「停止」
- 动画停止
- 显示中奖结果

**步骤 6**: (可选) 点击「重置」
- 重置抽奖状态
- 准备下一轮抽奖

### 3. 按钮状态说明

| 按钮 | 可用条件 | 说明 |
|------|---------|------|
| 开始抽奖 | 停止状态 | 启动抽奖动画和算法 |
| 暂停 | 抽奖进行中 | 暂停动画，不停止算法 |
| 停止 | 抽奖进行中/已暂停 | 停止动画，显示结果 |
| 重置 | 任意状态 | 重置为初始状态 |

### 4. 抽奖状态指示

| 状态 | 颜色 | 说明 |
|------|------|------|
| 已停止 | 灰色 | 初始状态，等待抽奖 |
| 抽奖进行中 | 绿色闪烁 | 正在抽奖 |
| 已暂停 | 黄色 | 动画暂停 |
| 抽奖完成 | 蓝色 | 已产生结果 |

## 🔧 常见操作示例

### 场景1: 公司年会抽奖（100人，5个奖品）

**准备工作**:
1. 准备Excel导入名单
2. 创建5个奖品（一等奖到五等奖）
3. 创建1个场次
4. 批量导入100人到场次
5. 添加5个奖品到场次

**抽奖流程**:
```
第一轮（一等奖）: 开始 → 停止 → 查看结果
第二轮（二等奖）: 重置 → 开始 → 停止 → 查看结果
第三轮（三等奖）: 重置 → 开始 → 停止 → 查看结果
...
```

### 场景2: 现场加抽环节

**临时添加奖品**:
1. 奖品管理 → 添加奖品
2. 抽奖组合 → 场次详情 → 奖品 → 添加奖品

**临时添加人员**:
1. 人员管理 → 单独添加或批量导入
2. 抽奖组合 → 场次详情 → 参与者 → 添加人员

### 场景3: 调整中奖概率

**设置权重**:
1. 人员管理 → 找到目标人员
2. 编辑 → 修改「中奖权重」
3. 权重越高，被抽中概率越大

**示例**:
- VIP用户：weight = 5（5倍概率）
- 普通用户：weight = 1（默认概率）

## ⚠️ 注意事项

### 数据安全
- 定期导出抽奖记录备份
- 删除人员前确认无进行中抽奖
- 敏感操作（如清空数据）需二次确认

### 抽奖公平性
- 抽奖开始前确认参与者列表正确
- 抽奖过程中不要修改参与者数据
- 黑名单用户在抽奖开始前设置完成

### 现场布置
- 建议使用Chrome或Edge浏览器
- 提前测试投影/投屏效果
- 准备备用电脑以防故障
- 网络环境稳定（有线网络优于无线）
