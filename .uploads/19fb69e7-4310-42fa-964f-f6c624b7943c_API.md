## 📋 API概览

### 基础信息

| 项目 | 说明 |
|------|------|
| 基础URL | `http://localhost:3002/api` |
| 通信协议 | HTTP REST + WebSocket |
| 数据格式 | JSON |
| 字符编码 | UTF-8 |
| 认证方式 | 无（局域网使用） |

### 响应格式

**成功响应**:
```json
{
  "success": true,
  "data": { ... }
}
```

**错误响应**:
```json
{
  "success": false,
  "error": "错误描述信息"
}
```

**带数据的成功响应**:
```json
{
  "id": 1,
  "name": "张三",
  "email": "zhangsan@example.com",
  "status": "未中奖"
}
```

## 👥 人员管理 API

### 基础路径: `/api/participants`

### 1. 获取人员列表

```
GET /api/participants
```

**查询参数**:

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| status | string | 否 | 筛选状态：已中奖/未中奖 |
| blacklist | number | 否 | 黑名单：0=正常, 1=黑名单 |
| search | string | 否 | 搜索关键词（姓名/邮箱/电话） |

**示例请求**:
```bash
curl http://localhost:3002/api/participants?status=未中奖&blacklist=0
```

**成功响应** (200):
```json
[
  {
    "id": 1,
    "name": "张三",
    "email": "zhangsan@example.com",
    "phone": "13800138000",
    "avatar": "",
    "status": "未中奖",
    "is_blacklisted": 0,
    "weight": 1,
    "created_at": "2026-04-03 10:00:00"
  },
  {
    "id": 2,
    "name": "李四",
    "email": "lisi@example.com",
    "phone": "13900139000",
    "avatar": "",
    "status": "已中奖",
    "is_blacklisted": 0,
    "weight": 1,
    "created_at": "2026-04-03 10:05:00"
  }
]
```

### 2. 获取单个人员

```
GET /api/participants/:id
```

**路径参数**:

| 参数 | 类型 | 说明 |
|------|------|------|
| id | number | 人员ID |

**示例请求**:
```bash
curl http://localhost:3002/api/participants/1
```

**成功响应** (200):
```json
{
  "id": 1,
  "name": "张三",
  "email": "zhangsan@example.com",
  "phone": "13800138000",
  "avatar": "",
  "status": "未中奖",
  "is_blacklisted": 0,
  "weight": 1,
  "created_at": "2026-04-03 10:00:00"
}
```

**错误响应** (404):
```json
{
  "success": false,
  "error": "人员不存在"
}
```

### 3. 创建人员

```
POST /api/participants
```

**请求体**:

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| name | string | ✅ | 姓名 |
| email | string | 否 | 邮箱 |
| phone | string | 否 | 电话 |
| weight | number | 否 | 中奖权重（默认1） |

**示例请求**:
```bash
curl -X POST http://localhost:3002/api/participants \
  -H "Content-Type: application/json" \
  -d '{"name": "王五", "email": "wangwu@example.com", "phone": "13700137000", "weight": 2}'
```

**成功响应** (201):
```json
{
  "id": 5,
  "name": "王五",
  "email": "wangwu@example.com",
  "phone": "13700137000",
  "status": "未中奖",
  "is_blacklisted": 0,
  "weight": 2
}
```

**错误响应** (400):
```json
{
  "success": false,
  "error": "姓名不能为空"
}
```

**错误响应** (409 - 重复):
```json
{
  "success": false,
  "error": "该姓名和邮箱组合已存在",
  "existing_id": 1
}
```

### 4. 更新人员

```
PUT /api/participants/:id
```

**路径参数**:

| 参数 | 类型 | 说明 |
|------|------|------|
| id | number | 人员ID |

**请求体**:

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| name | string | 否 | 姓名 |
| email | string | 否 | 邮箱 |
| phone | string | 否 | 电话 |
| weight | number | 否 | 中奖权重 |

**示例请求**:
```bash
curl -X PUT http://localhost:3002/api/participants/1 \
  -H "Content-Type: application/json" \
  -d '{"name": "张三(修改)", "weight": 3}'
```

**成功响应** (200):
```json
{
  "id": 1,
  "name": "张三(修改)",
  "email": "zhangsan@example.com",
  "phone": "13800138000",
  "status": "未中奖",
  "is_blacklisted": 0,
  "weight": 3
}
```

### 5. 删除人员

```
DELETE /api/participants/:id
```

**示例请求**:
```bash
curl -X DELETE http://localhost:3002/api/participants/5
```

**成功响应** (200):
```json
{
  "success": true,
  "message": "人员已删除"
}
```

### 6. 更新人员状态

```
PUT /api/participants/:id/status
```

**请求体**:

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| status | string | ✅ | 状态：已中奖/未中奖 |

**示例请求**:
```bash
curl -X PUT http://localhost:3002/api/participants/1/status \
  -H "Content-Type: application/json" \
  -d '{"status": "已中奖"}'
```

**成功响应** (200):
```json
{
  "success": true,
  "message": "状态已更新"
}
```

### 7. 设置黑名单

```
PUT /api/participants/:id/blacklist
```

**请求体**:

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| is_blacklisted | number | ✅ | 0=取消黑名单, 1=加入黑名单 |

**示例请求**:
```bash
curl -X PUT http://localhost:3002/api/participants/1/blacklist \
  -H "Content-Type: application/json" \
  -d '{"is_blacklisted": 1}'
```

**成功响应** (200):
```json
{
  "success": true,
  "message": "黑名单状态已更新"
}
```

### 8. 批量导入

```
POST /api/participants/import
```

**请求体**: FormData

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| file | File | ✅ | Excel文件(.xlsx/.xls) |
| session_id | number | 否 | 关联的场次ID |

**示例请求**:
```bash
curl -X POST http://localhost:3002/api/participants/import \
  -F "file=@participants.xlsx" \
  -F "session_id=1"
```

**成功响应** (200):
```json
{
  "success": true,
  "imported": 95,
  "skipped": 5,
  "message": "导入完成"
}
```

**Excel文件格式**:
| name | email | phone |
|------|-------|-------|
| 张三 | zhangsan@example.com | 13800138000 |
| 李四 | lisi@example.com | 13900139000 |

## 🔌 WebSocket 事件

### 1. 客户端 → 服务器事件

#### display:status

**描述**: 展示页定期报告状态

**数据格式**:
```javascript
{
  isDrawing: boolean,      // 是否正在抽奖
  isWinner: boolean,      // 是否刚中奖项
  currentSessionId: number,// 当前场次ID
  animationType: string,   // 动画类型
  connectionTime: number   // 连接时长(秒)
}
```

**触发时机**: 定期（每5秒）或状态变化时

#### draw:start

**描述**: 请求开始抽奖

**数据格式**:
```javascript
{
  session_id: number,  // 场次ID
  prize_id: number    // 奖品ID
}
```

#### draw:pause / draw:stop / draw:reset

**描述**: 控制指令

**数据格式**: 无

### 2. 服务器 → 客户端事件

#### draw:start

**描述**: 抽奖开始

**数据格式**:
```javascript
{
  session_id: number,
  prize_id: number,
  prize_name: string,     // 奖品名称
  total_participants: number, // 总参与人数
  timestamp: number
}
```

#### draw:pause

**描述**: 抽奖暂停

**数据格式**: 无

#### draw:stop

**描述**: 抽奖停止

**数据格式**: 无

#### draw:result

**描述**: 抽奖结果

**数据格式**:
```javascript
{
  winner: {
    id: number,
    name: string,
    email: string,
    phone: string,
    avatar: string
  },
  prize: {
    id: number,
    name: string,
    description: string,
    image: string
  },
```

## 📝 使用示例

### JavaScript (Fetch API)

```javascript
// 获取参与者列表
async function getParticipants() {
  const response = await fetch('http://localhost:3002/api/participants');
  const data = await response.json();
  return data;
}

// 创建人员
async function createParticipant(name, email) {
  const response = await fetch('http://localhost:3002/api/participants', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ name, email })
  });
  return response.json();
}
```

### Python (requests)

```python
import requests

# 获取参与者列表
response = requests.get('http://localhost:3002/api/participants')
participants = response.json()

# 创建人员
response = requests.post(
    'http://localhost:3002/api/participants',
    json={'name': '张三', 'email': 'zhangsan@example.com'}
)
print(response.json())
```

### cURL

```bash
# 完整抽奖流程
# 1. 创建场次

  -H "Content-Type: application/json" \
  -d '{"name": "测试场次"}'

# 2. 添加参与者
curl -X POST http://localhost:3002/api/sessions/1/participants \
  -H "Content-Type: application/json" \

  -H "Content-Type: application/json" \

curl -X POST http://localhost:3002/api/sessions/1/draw \
  -H "Content-Type: application/json" \
  -d '{"prize_id": 1}'
```
