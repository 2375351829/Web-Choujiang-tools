# 抽奖系统 用户手册
**版本**: v1.0.0 | **日期**: 2026-04-03

## 1. 快速开始

### 1.1 系统访问
| 页面 | 地址 | 用途 | 使用者 |
| 管理后台 | http://localhost:3000/admin | 管理数据、控制抽奖 | 管理员 |
| 抽奖展示 | http://localhost:3000/display | 大屏动画展示 | 观众 |

### 1.2 首次使用流程
```
启动系统 → 导入人员 → 创建奖品 → 创建场次 → 配置场次 → 开始抽奖
 (1分钟)   (3分钟)   (2分钟)   (1分钟)   (2分钟)   (随时)
```

### 1.3 系统要求
- 浏览器：Chrome 80+ / Edge 80+ 推荐
- 网络：管理后台和展示页面需能访问后端服务
- 显示：展示页面建议全屏模式（F11），推荐分辨率1920x1080

## 2. 管理后台使用指南

### 2.1 界面介绍
管理后台采用左侧菜单 + 右侧内容的布局：
- 顶部导航栏：系统名称、刷新按钮、全屏按钮
- 左侧菜单：控制台、人员管理、奖品管理、场次管理、系统配置
- 右侧内容：根据菜单选择显示对应页面

### 2.2 控制台（抽奖控制）

#### 界面说明
控制台是抽奖操作的核心页面，包含：
- **抽奖控制面板**：场次选择、奖品选择、操作按钮
- **状态面板**：连接状态、抽奖状态、已中奖人数
- **活动日志**：最近操作记录

#### 操作步骤

**步骤1：选择场次和奖品**
- 在场次下拉框选择要抽奖的场次
- 在奖品下拉框选择要抽取的奖品
- 系统会显示该奖品的剩余数量

**步骤2：开始抽奖**
- 点击「开始抽奖」按钮
- 展示大屏开始播放动画
- 按钮状态变化：开始按钮变灰，暂停/停止按钮可用

**步骤3：暂停/停止**
- 点击「暂停」可暂停动画（可选）
- 点击「停止」执行抽奖并显示结果
- 中奖者信息显示在结果区域

**步骤4：重置**
- 点击「重置」恢复初始状态
- 可选择下一个奖品继续抽奖

#### 按钮状态说明
| 按钮 | 可用条件 | 功能 |
| 开始抽奖 | 停止状态 | 启动抽奖动画 |
| 暂停 | 抽奖进行中 | 暂停动画 |
| 停止 | 进行中/已暂停 | 执行抽奖，显示结果 |
| 重置 | 任意状态 | 恢复初始状态 |

#### 状态指示
| 状态 | 颜色 | 含义 |
| 灰色 | 灰色圆点 | 初始/停止状态 |
| 绿色闪烁 | 绿色圆点 | 抽奖进行中 |
| 黄色 | 黄色圆点 | 已暂停 |
| 蓝色 | 蓝色圆点 | 已出结果 |

### 2.3 人员管理

#### 添加人员
1. 点击左侧菜单「人员管理」
2. 点击「添加人员」按钮
3. 填写信息：
   - 姓名（必填）
   - 邮箱（选填）
   - 电话（选填）
   - 中奖权重（默认1，数值越大被抽中概率越高）
4. 点击「确认添加」

#### 批量导入
1. 准备Excel文件，格式如下：
   | name | email | phone |
   | 张三 | zhangsan@email.com | 13800138000 |
   | 李四 | lisi@email.com | 13900139000 |
2. 点击「批量导入」按钮
3. 选择Excel文件
4. 等待导入完成，查看结果

#### 编辑/删除人员
- 编辑：点击操作列的「编辑」按钮，修改后保存
- 删除：点击操作列的「删除」按钮，确认后删除

#### 黑名单管理
- 点击人员列表中的「黑名单」开关
- 加入黑名单的人员不参与抽奖
- 黑名单状态以红色标签显示

#### 搜索筛选
- 按状态筛选：全部/已中奖/未中奖
- 按黑名单筛选：全部/正常/黑名单
- 关键词搜索：输入姓名/邮箱/电话搜索

### 2.4 奖品管理

#### 添加奖品
1. 点击左侧菜单「奖品管理」
2. 点击「添加奖品」按钮
3. 填写信息：
   - 奖品名称（必填）
   - 奖品描述（选填）
   - 奖品图片（选填，支持jpg/png/gif，最大10MB）
   - 可抽次数（默认1，即该奖品可被抽中几次）
4. 点击「确认添加」

#### 编辑/删除奖品
- 编辑：点击奖品卡片的「编辑」按钮
- 删除：点击「删除」按钮，确认后删除

### 2.5 场次管理

#### 创建场次
1. 点击左侧菜单「场次管理」
2. 点击「创建场次」按钮
3. 填写场次名称（必填）和描述（选填）
4. 点击「确认创建」

#### 配置场次
进入场次详情页，包含三个标签页：

**参与者标签**：
- 点击「添加参与者」
- 勾选要添加的人员
- 或点击「批量导入」直接导入到场次

**奖品标签**：
- 点击「添加奖品」
- 选择奖品并设置数量
- 数量表示该场次中此奖品可被抽中几次

**结果标签**：
- 查看该场次所有中奖记录
- 包含中奖者、奖品、时间信息

### 2.6 系统配置

#### 主题切换
- 提供6种预设主题：暗夜黑、简约白、海洋蓝、森林绿、落日橙、梦幻紫
- 点击色块即可切换，实时预览

#### 背景设置
- 渐变背景：选择起始色、终止色、角度
- 纯色背景：选择一个颜色
- 图片背景：上传背景图片

#### 动画设置
- 动画类型：网格(Grid)、球体(Sphere)、列表(List)
- 轮播速度：滑块调节（2000-5000ms）

#### 显示设置
- 活动标题：自定义展示页面标题
- 显示开关：控制是否显示参与者区域、奖品区域、中奖区域

## 3. 抽奖展示页面使用指南

### 3.1 访问方式
- 地址：http://localhost:3000/display
- 建议使用全屏模式（F11）
- 建议连接投影仪或大屏幕

### 3.2 页面布局
展示页面从上到下包含：
1. **活动标题**：显示配置的活动名称
2. **参与者轮播**：参与者姓名滚动展示（可配置隐藏）
3. **抽奖动画区域**：核心动画展示区域
4. **中奖结果**：中奖时显示中奖者信息
5. **中奖名单**：已中奖者滚动列表（可配置隐藏）
6. **连接状态**：显示与服务器连接状态

### 3.3 动画模式说明
| 模式 | 效果 | 适用场景 |
| 网格 | 3x3方格高亮跳动 | 参与者较少(<=9人) |
| 球体 | 3D球体旋转 | 参与者较多 |
| 列表 | 纵向列表滚动 | 大量参与者 |

### 3.4 中奖展示效果
中奖时展示页面会：
1. 播放彩带飘落特效
2. 中奖者姓名放大显示
3. 奖品名称同步展示
4. 中奖者添加到下方名单

### 3.5 连接状态
| 状态 | 显示 | 说明 |
| 已连接 | 绿色 | 正常状态 |
| 未连接 | 红色 | 检查后端服务是否运行 |
| 重连中 | 黄色 | 网络异常，正在重连 |

## 4. 常见问题

### 4.1 管理后台问题
**Q: 页面空白或加载失败？**
A: 检查后端服务是否运行，检查浏览器控制台(F12)是否有错误

**Q: 点击开始抽奖无反应？**
A: 检查是否选择了场次和奖品，检查WebSocket连接状态

**Q: 数据导入失败？**
A: 确认Excel格式正确（必须包含name列），文件为.xlsx或.xls格式

### 4.2 展示页面问题
**Q: 显示"未连接服务器"？**
A: 检查后端服务是否运行，确认访问地址正确

**Q: 动画卡顿？**
A: 关闭其他程序，尝试使用列表动画模式，降低参与者显示数量

**Q: 背景不显示？**
A: 检查配置是否正确，清除浏览器缓存(Ctrl+Shift+Delete)

### 4.3 抽奖问题
**Q: 提示"该奖品已经抽完"？**
A: 该奖品已达到设定的可抽次数，可在场次配置中增加数量

**Q: 提示"没有可参与抽奖的人员"？**
A: 所有参与者已中奖或被加入黑名单

**Q: 中奖结果与预期不符？**
A: 检查参与者权重配置，权重越高被抽中概率越大

## 6. 最佳实践

### 6.1 抽奖前准备
1. 提前导入所有参与者名单
2. 创建所有奖品并设置正确数量
3. 创建场次并关联参与者和奖品
4. 测试一次抽奖流程确认正常
5. 提前打开展示页面并确认连接正常

### 6.2 现场抽奖建议
1. 使用有线网络连接（比WiFi更稳定）
2. 展示页面使用全屏模式
3. 准备备用电脑
4. 抽奖前确认参与者名单正确
5. 抽奖过程中不要修改数据

### 6.3 抽奖后处理
1. 导出中奖结果备份
2. 备份数据库文件
3. 记录抽奖日志
