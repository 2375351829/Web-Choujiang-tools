# 抽奖系统 UI/UX设计文档
**版本**: v1.0.0 | **日期**: 2026-04-03

## 1. 设计概述

### 1.1 设计理念
- 管理后台：功能优先，操作高效，信息密度适中
- 抽奖展示：视觉冲击，动画流畅，大屏适配

### 1.2 设计规范
- 设计工具：Figma / Sketch
- 基准分辨率：1920x1080
- 色彩模式：sRGB
- 字体基准：14px (管理后台) / 24px (展示页面)

### 2.2 页面设计

#### 2.2.1 控制台页面 (Dashboard)
- 左侧：抽奖控制面板（场次选择、奖品选择、开始/暂停/停止/重置按钮）
- 右侧上：当前状态卡片（连接状态、抽奖状态、已中奖人数）
- 右侧下：最近活动日志
- 按钮状态：根据抽奖状态机控制按钮可用性

#### 2.2.2 人员管理页面
- 顶部：搜索栏 + 筛选器 + 添加按钮 + 批量导入按钮
- 中部：数据表格（序号、姓名、邮箱、电话、状态、黑名单、权重、操作）
- 操作列：编辑、删除、黑名单切换
- 分页：底部分页器

#### 2.2.3 奖品管理页面
- 卡片式布局展示奖品列表
- 每个卡片包含：奖品图片、名称、描述、数量、操作按钮
- 添加/编辑使用弹窗表单

#### 2.2.4 场次管理页面
- 场次列表（卡片或表格形式）
- 点击进入场次详情页
- 场次详情页使用Tab切换：参与者 | 奖品 | 结果

#### 2.2.5 系统配置页面
- 左侧：主题选择（6种预设主题的色块预览）
- 右侧：配置表单
  - 背景设置（类型选择 + 对应配置项）
  - 动画设置（类型选择 + 速度滑块）
  - 显示设置（开关组件）
  - 标题设置（文本输入）

### 2.3 通用组件规范
#### 表格组件
- 斑马纹、固定表头、排序
- 操作列固定在右侧
- 空状态提示

#### 表单弹窗
- 宽度：500px
- 必填字段标记红色星号
- 表单验证实时提示
- 底部操作按钮：取消 + 确认

#### 状态标签
- 未中奖：灰色标签
- 已中奖：绿色标签
- 黑名单：红色标签
- 场次状态：蓝色(未开始)/绿色(进行中)/灰色(已结束)

### 3.2 动画设计

#### 3.2.1 Grid动画（网格模式）
- 3x3网格布局，每格显示一个参与者姓名
- 当前高亮格子发光效果（box-shadow + scale）
- 高亮在格子间快速跳动
- 中奖时：目标格子放大 + 金色边框 + 其他格子淡出

#### 3.2.2 Sphere动画（球体模式）
- 3D球体效果，参与者姓名分布在球面
- 球体持续旋转
- 当前高亮姓名放大 + 发光
- 中奖时：球体停止旋转，中奖者姓名居中放大

#### 3.2.3 List动画（列表模式）
- 纵向列表展示参与者
- 当前高亮项背景色变化 + 左侧指示器
- 高亮在列表中快速滚动
- 中奖时：列表停止，中奖项居中 + 特殊样式

#### 3.2.4 中奖特效
- 彩带飘落效果（canvas-confetti）
- 中奖者姓名弹出放大动画
- 奖品名称渐入显示
- 背景闪光效果

### 3.3 响应式设计

#### 断点设置
| 断点 | 宽度 | 字体基准 | 网格大小 |
| mobile | <640px | 14px | 2x2 |
| tablet | 640-1024px | 18px | 3x3 |
| desktop | 1024-1920px | 24px | 3x3 |
| large | >1920px | 28px | 4x4 |

#### 响应式字体
使用CSS clamp()实现：
```css
.title { font-size: clamp(28px, 5vw, 72px); }
.name { font-size: clamp(14px, 2vw, 24px); }
```

## 5. 交互设计

### 5.1 抽奖操作交互
- 开始按钮：点击后变为不可用，暂停和停止按钮可用
- 暂停按钮：仅在抽奖进行中可用
- 停止按钮：点击后执行抽奖算法，展示结果
- 重置按钮：任意状态可用，重置为初始状态

### 5.2 数据操作交互
- 创建/编辑：使用弹窗表单，支持ESC关闭
- 删除：二次确认弹窗
- 批量导入：拖拽上传区域 + 进度条
- 筛选：实时筛选，无需提交

### 5.3 反馈设计
- 成功操作：绿色提示消息（顶部）
- 失败操作：红色提示消息（顶部）
- 加载状态：按钮loading + 骨架屏
- 空状态：插图 + 文字提示

### 5.4 快捷键
| 快捷键 | 功能 | 适用页面 |
| F11 | 全屏切换 | 全局 |
| Escape | 关闭弹窗 | 管理后台 |
| Space | 开始/停止抽奖 | 控制台（焦点在控制区时） |
