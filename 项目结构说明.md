# 项目结构说明

## 1. 项目概述

本项目是一个基于Node.js和Vue.js开发的网页抽奖工具，用于管理和执行抽奖活动。项目使用Express作为后端框架，SQLite作为数据库，Vue 3作为前端框架，支持实时抽奖功能。

## 2. 目录结构

```
├── routes/             # API路由目录
│   ├── console.js      # 控制台相关API
│   ├── participants.js # 参与者相关API
│   ├── prizes.js       # 奖品相关API
│   └── sessions.js     # 抽奖场次相关API
├── src/                # 前端源码目录
│   ├── App.vue         # 主Vue组件
│   └── main.js         # 前端入口
├── uploads/            # 文件上传目录（存放图片和Excel文件）
├── database.db         # SQLite数据库文件
├── db.js               # 数据库初始化和连接
├── server.js           # 主服务器文件
├── package.json        # 项目依赖和脚本
├── package-lock.json   # 依赖版本锁定文件
├── vite.config.js      # Vite配置文件
├── index.html          # 前端入口HTML
└── README.md           # 项目说明文件
```

## 3. 核心文件说明

### 3.1 后端文件

#### server.js
- 主服务器文件，负责启动Express服务器和Socket.io服务
- 配置文件上传功能
- 注册API路由
- 处理Socket.io实时通信
- 提供Excel批量导入功能
- 提供背景图片上传接口

#### db.js
- 数据库初始化和连接文件
- 创建数据库表结构
- 添加索引以提高查询性能

#### routes/
- **console.js**：控制台相关API，用于控制抽奖过程
- **participants.js**：参与者管理API，包括增删改查操作
- **prizes.js**：奖品管理API，包括增删改查操作
- **sessions.js**：抽奖场次管理API，包括创建、编辑、删除场次等操作

### 3.2 前端文件

#### src/App.vue
- 主Vue组件，包含整个应用的布局和主要功能

#### src/main.js
- 前端入口文件，初始化Vue应用

### 3.3 配置文件

#### package.json
- 项目依赖和脚本配置
- 定义了项目的名称、版本、描述等信息
- 列出了项目依赖的第三方库

#### vite.config.js
- Vite构建工具配置

## 4. 数据库表结构

### 4.1 participants（参与者表）
- `id`：主键，自增
- `name`：参与者姓名
- `email`：参与者邮箱
- `phone`：参与者电话
- `status`：状态（未中奖/已中奖）
- `is_blacklisted`：是否在黑名单中
- `created_at`：创建时间

### 4.2 lottery_results（抽奖结果表）
- `id`：主键，自增
- `participant_id`：参与者ID（外键）
- `prize`：奖品名称
- `drawn_at`：抽奖时间

### 4.3 prizes（奖品表）
- `id`：主键，自增
- `name`：奖品名称
- `description`：奖品描述
- `image`：奖品图片
- `weight`：权重
- `draw_count`：抽取数量
- `created_at`：创建时间

### 4.4 lottery_sessions（抽奖场次表）
- `id`：主键，自增
- `name`：场次名称
- `description`：场次描述
- `status`：状态（未开始/进行中/已结束）
- `created_at`：创建时间
- `updated_at`：更新时间

### 4.5 session_prizes（场次-奖品关联表）
- `id`：主键，自增
- `session_id`：场次ID（外键）
- `prize_id`：奖品ID（外键）
- `quantity`：数量

### 4.6 session_participants（场次-参与者关联表）
- `id`：主键，自增
- `session_id`：场次ID（外键）
- `participant_id`：参与者ID（外键）

### 4.7 session_results（场次抽奖结果表）
- `id`：主键，自增
- `session_id`：场次ID（外键）
- `participant_id`：参与者ID（外键）
- `prize_id`：奖品ID（外键）
- `drawn_at`：抽奖时间

## 5. 项目功能模块

### 5.1 参与者管理
- 添加、编辑、删除参与者
- 批量导入参与者（Excel）
- 查看参与者列表
- 将参与者加入黑名单

### 5.2 奖品管理
- 添加、编辑、删除奖品
- 上传奖品图片
- 设置奖品权重和抽取数量

### 5.3 抽奖场次管理
- 创建、编辑、删除抽奖场次
- 为场次分配奖品
- 为场次分配参与者

### 5.4 抽奖控制
- 开始抽奖
- 暂停抽奖
- 停止抽奖
- 重置抽奖

### 5.5 实时通信
- 使用Socket.io实现实时抽奖效果
- 多端同步抽奖状态

## 6. 技术栈

- **后端**：Node.js, Express, SQLite, Socket.io
- **前端**：Vue 3, Element Plus
- **构建工具**：Vite
- **其他**：Multer（文件上传）, XLSX（Excel处理）
